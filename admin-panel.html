Connected to GitHub — changes save directly to your repository
1

Create a GitHub Personal Access Token

Go to github.com/settings/tokens/new, give it a name like "Patricia Admin", tick repo scope, and click Generate.

2

Enter your details below

Your token is saved only in your browser's local storage — it never leaves your device except to communicate directly with GitHub's API over HTTPS.

3

Test the connection

Click "Connect & Test" — the portal will verify your credentials. Once connected, adding or deleting a property updates your live website automatically.
